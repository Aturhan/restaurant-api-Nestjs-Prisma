## Description

With the Restaurant API app, users will be able to order food from the restaurant.

## Dependencies
- Prisma
- JWT
- PostgreSQL
- Passport
- Validator


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
